# 📰 Fake News Detection System

## 📌 Project Overview

The **Fake News Detection System** is a Machine Learning based web application that detects whether a news article is real or fake.

The system analyzes news content using Natural Language Processing (NLP) and Machine Learning techniques to improve news reliability and reduce misinformation.

---

# 🚀 Features

✅ Detects fake and real news articles

✅ NLP-based text preprocessing

✅ Machine Learning prediction model

✅ User-friendly web interface

✅ Fast and accurate predictions



